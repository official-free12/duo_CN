# 多语言区块链销售和拍卖交易平台

ICOMAX是一个先进、多语言和稳定的区块链销售和拍卖平台。您可以使用法定货币和多种加密货币创建和出售区块链。拍卖、聊天也是这个平台的主要功能。这个应用程序基于Laravel example.com<br>演示站:example.com/<br>管理员登录链接:example.com/admin<br>管理员:admin 密码:admin<br>
